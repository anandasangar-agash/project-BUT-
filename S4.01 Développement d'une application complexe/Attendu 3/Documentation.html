Documentation PHP

Controller

Ceci est la classe de contrôleur de base que les autres contrôleurs étendent. Elle inclut des fonctionnalités communes requises par tous les contrôleurs.

__construct

public function __construct()

Initialise le contrôleur.

index

public function index()

La fonction par défaut appelée lorsqu'aucune fonction spécifique n'est demandée.

loadModel

public function loadModel($modelName)

Charge le modèle spécifié.

  • Paramètres : string $modelName - Le nom du modèle à charger.
  • Retour : object - Le modèle chargé.

renderView

public function renderView($viewName, $data = [])

Affiche la vue spécifiée avec les données fournies.

  • Paramètres :
    • string $viewName - Le nom de la vue à afficher.
    • array $data - Les données à passer à la vue.
  • Retour : void
